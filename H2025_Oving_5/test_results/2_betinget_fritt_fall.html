Introduksjon:

Vi fortsetter se p fallskjermhopperen med i fritt fall. I oppgave $1$ introduserte vi konseptet terminalhastighet som er den maksimale hastigheten det er mulig oppn i fritt fall, og defineres som hastigheten $v$ der luftmotstanden og tyngdekraften utligner hverandre. I denne oppgaven skal vi ta for oss en litt forenklet matematisk modell for hvordan hastigheten utvikler seg over tid. Vi antar at det er to distinkte tilfeller:

  1. Perioden fr hopperen nr sin terminalfart
  2. Perioden nr hopperen beveger seg i terminalfart

Perioden fr terminalfart:
Ved $t=0$ antar vi at farten er 0. Deretter er det en liner sammenheng mellom tiden og hastigheten, denne sammenhengen er gitt ved forenklede ligningen:

$ v(t) = t \cdot g$

Der $g = 9.81$ er tyngdeakselerasjonen og $t$ er tiden siden hoppet startet.

Perioden etter terminalhastighet er oppndd:
For enkelhets skyld antar vi at alle hoppere, uavhengig av vekt, har en terminalhastighet $v_{\text{term}} = 50 \text{ m/s}$. Terminalhastigheten oppns p et tisplunkt $t=t_a$, hvorp hastigheten til hopperen etter dette tidspunktet holder seg konstant, og er gitt ved ligningen:

$v(t) = 50$

Ved kombinere disse to overnevnte tilfellene, kan vi skrive hastigheten matematisk som en betinget funksjon:

$$v(t) = \begin{cases} g\cdot t, & t \leq t_a \\ 50\text{ m/s}, & t > t_a \end{cases} \tag{2.1}$$

hvor tidspunktet $t_a = \frac{50 \text{ m/s}}{g} \approx 5.1 \text{ s}$.

Betingede funksjoner i Python

I ving 4 s vi p bruk av if-setinger til regne p betingede funksjoner (som f.eks. abs(x)). Absoluttverdi-funksjonen $\text{abs}(x)$ kan skrives som en betinget funksjon som flger:

$ \text{abs}(x) = \begin{cases} x, \text{ if } x \geq 0\\ -x, \text{ if } x < 0 \end{cases}$

I Python kan vi hndtere dette ved underske hvilken input som gis til funksjonen, og returnere den riktige verdien, slik det er gjort i eksempelet nedenfor.

In[1]:
def abs_tall(x):
    if x > 0:
        return x
    else:
        return -x
        
# Dersom vi gir en positiv verdi som input, forventer samme positive verdi som output
print(f"{abs_tall(2) = }")
# Dersom vi gir en negativ verdi som input, forventer vi at fortenget er blitt omgjort
print(f"{abs_tall(-2) = }")

abs_tall(2) = 2
abs_tall(-2) = 2

Betingede funksjoner med liste/array som input

Som kjent er if-setninger drlig egnet til hndtere arrays, og dersom vi nsker benytte funksjonen over til regne ut $\text{abs}(x)$ for alle verdiene i en array m, vi benytte en lkke. Nedenfor ser du funksjonen abs_liste som kan brukes med lister/arrays som input.

In[4]:
def abs_liste(x_values):
    y_values = []  # Forbered en tom liste som skal holde alle verdiene y = |x|
    for x in x_values:
        y = abs_tall(x)
        y_values.append(y)
    return y_values

Forklaringen av funksjonens innhold er som flger:

  1. Lag en "tom" liste y_list. Denne skal etter hvert inneholde funksjonsverdiene $f(x)$ for alle verdiene i x_array nr lkken er ferdig.
    • Vi bruker en liste og ikke en array for kunne dra nytte av muligheten til legge til elementer underveis.
  2. Begynn en for-lkke som itererer over alle elementer i arrayen x_array.
    • For hver av verdiene x, finner vi den riktige utgangsverdien y ved kalle den tidligere funksjonen abs_tall.
    • Vi skal dog ikke returnere utregnede verdier y, men istedenfor skrive de til listen vi lagde i steg 1.
  3. Returner til slutt hele listen y_list konvertert til en numpy-array.

def v_liste(t):
    g = 9.81
    v_term = 50
    ### BEGIN SOLUTION
    t_a = v_term/g
    y = []
    for i in range(len(t)):
        if t[i] >= t_a:
            y.append(v_term)
        else:
            y.append(t[i]*g)
    return y
    ### END SOLUTION
